# 设计规范文档 (Design System)

## 1. 智能画布规范 (Smart Canvas Specs)

为了保证时间轴的整洁和统一，AI 生成内容的容器需要遵循严格的尺寸限制。

*   **容器尺寸**:
    *   **宽度**: 100% (相对于父容器 `TimelineNode`)。
    *   **最大宽度**: 移动端约 `350px`，桌面端约 `400px`。
    *   **高度策略**: 
        *   **固定高度**: 推荐 `500px` 或 `600px`，确保每个卡片视觉量级一致，AI 需在此范围内布局（支持内部滚动 `overflow-y: auto` 或 `hidden`）。
        *   **宽高比**: 推荐 `3:4` (Portrait) 或 `1:1` (Square)。
    
*   **样式隔离**:
    *   画布内部的背景色、字体、边框由 AI 决定。
    *   画布外部必须有统一的圆角 (`rounded-2xl`) 和阴影 (`shadow-md`)，由外层 React 组件控制，AI 只需要填充内部。

## 2. 全局色彩 (App Shell)

*   **背景**: `bg-slate-50` 或 `bg-pink-50` (极淡)。
*   **时间轴**: 
    *   中轴线颜色: `border-pink-200` (实线或点状)。
    *   日期徽章: `bg-white text-pink-500 font-bold shadow-sm`.

## 3. 字体 (Typography)

*   **外层 UI**: 使用系统无衬线字体，保持清晰导航。
*   **画布内**: 允许 AI 引用 Google Fonts (需在 Prompt 中允许) 或使用通用字体族，以表现不同的艺术风格（手写体、衬线体等）。

## 4. 交互反馈
*   **生成中**: 展示一个加载动画（例如：一颗跳动的心，或“AI 正在设计中...”的文字）。
*   **生成后**: 使用 `iframe` 或 Shadow DOM 渲染，避免布局抖动。
